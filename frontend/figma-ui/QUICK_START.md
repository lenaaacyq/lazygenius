# ⚡ 快速启动指南

> 3 分钟快速启动 LazyGenius 应用

## 🚀 三步启动

### 1️⃣ 安装依赖（首次运行）

```bash
npm install
```

**预计耗时**: 1-2 分钟

### 2️⃣ 启动开发服务器

```bash
npm run dev
```

**预计耗时**: 5-10 秒

### 3️⃣ 打开浏览器

自动打开：`http://localhost:5173`

或手动访问上述地址 ✅

---

## 📱 手机访问

### 同一 WiFi 下访问

启动后终端会显示：

```bash
➜ Local:   http://localhost:5173/
➜ Network: http://192.168.x.x:5173/  👈 用这个地址
```

用手机浏览器访问 Network 地址即可！

---

## 🎮 快速体验

启动成功后，立即尝试：

1. **👆 滑动卡片**
   - ← 左滑 = 归档
   - → 右滑 = 保留（紫绿高光）

2. **🔄 切换模式**
   - 点击顶部：混合 / 新卡 / 复习

3. **➕ 添加卡片**
   - 点击紫色「添加新链接」按钮
   - 输入任意网址
   - 观看生成动画

4. **📖 查看详情**
   - 点击卡片
   - 进入详情页

## 🐛 遇到问题？

### 端口被占用

```bash
# 杀掉占用进程
lsof -ti:5173 | xargs kill -9

# 或使用其他端口
npm run dev -- --port 3000
```

### 样式不生效

```bash
# 重启开发服务器
# Ctrl+C 停止，然后重新运行
npm run dev
```
